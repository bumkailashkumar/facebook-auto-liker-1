# Facebook Auto Liker

### version 0.3.1
Facebook likes feeds, likes stories, send message birthday

### Установка

 Python: 
```
pip install selenium
```
Windows:
```
downloads .exe files
```
### Совместимость
* Python 3.7+
* Selenium 3.4.0+
* Chrome driver 84.0.4147.105
* Chrome 84.0+ (for other version: [download](https://chromedriver.chromium.org/))
* Русский язык Facebook
