# Facebook Auto Liker
version 0.3.1
Facebook likes feeds, likes stories, send message birthday

### Установка

 Python: 
```
pip install selenium
pip install pyautogui
pip install pywin32
pip install opencv-python
```
Windows:
```
downloads .exe files
```
### Совместимость
* Python 3.7+
* Selenium 3.4.0+
* Chrome 81.1+
